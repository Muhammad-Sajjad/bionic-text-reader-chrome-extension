BIONIC TEXT READER EXTENSION ICONS

This directory contains the required icons for the Chrome extension.

REQUIRED ICONS:
- icon16.svg (16x16 pixels)
- icon48.svg (48x48 pixels)
- icon128.svg (128x128 pixels)

ACTIVE STATE ICONS:
- icon16-active.svg (16x16 pixels)
- icon48-active.svg (48x48 pixels)
- icon128-active.svg (128x128 pixels)

SVG VERSIONS:
The extension now uses SVG icons instead of PNG icons to avoid decoding issues.
The active state icons use a different color (#ff5722) to indicate when bionic text is enabled.

NOTE:
If you encounter any issues with SVG icons, you may need to convert them to PNG format.
The manifest.json and background.js files have been updated to use SVG icons.
