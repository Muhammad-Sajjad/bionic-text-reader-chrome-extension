BIONIC TEXT READER EXTENSION ICONS

This directory contains the required icons for the Chrome extension.

SVG ICONS:
- icon16.svg (16x16 pixels)
- icon48.svg (48x48 pixels)
- icon128.svg (128x128 pixels)

ACTIVE STATE ICONS:
- icon16-active.svg (16x16 pixels)
- icon48-active.svg (48x48 pixels)
- icon128-active.svg (128x128 pixels)

The active state icons are used when the bionic text feature is enabled.

NOTE:
The extension now uses SVG format for all icons. The PNG files are kept for reference but are no longer used.
The manifest.json and background.js files have been updated to reference the SVG files.
