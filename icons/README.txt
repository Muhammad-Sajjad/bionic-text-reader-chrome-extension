BIONIC TEXT READER EXTENSION ICONS

This directory contains the required icons for the Chrome extension.

REQUIRED ICONS:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

SVG VERSIONS:
The directory also includes SVG versions of the icons:
- icon16.svg
- icon48.svg
- icon128.svg

NOTE:
The manifest.json file requires these icons to be in PNG format. The extension will not work properly without these icons.
